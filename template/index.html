Hyper Cam Team

Senior Design Project



Summary

The goal of this project is to make surveillance cameras more user friendly by utilizing object detection and machining learning to review footage for the user. The team will deliver a product that allows the user to configure a search entry to enable the camera to locate specific targets.

This approach is new to the market because object detection and classification is hardware intensive and costly. As a result, companies have not seen the need to adopt it. The proposed product takes the shortcomings of processing images via a server or onboard the camera by implementing a two-step process. This enables the camera to perform an initial-filter of the captured frames before sending the frames over. This product is targeted towards companies, stores, and communities that currently have a large network of cameras that need to be reviewed. The product benefits them the most because they are able to autonomously review lots of footage but can still reduce their network traffic.
